# 서울 15억 미만 아파트 지도

서울 25개 자치구의 실거래가 ≤ 15억원인 아파트 단지 9,268개를 Leaflet 지도에 시각화.

## 라이브 데모
GitHub Pages 배포 후: https://USERNAME.github.io/REPO/

## 기능
- 가격(0~15억) · 전용면적 · 세대수 · 자치구 · 거래 시점(6개월/1년/2년/5년) 필터
- 원 색상 = 가격대 (초록 → 빨강), 원 크기 = 세대수
- 단지 클릭 시 실거래 최고가, 거래일자, 층수, 전세, 공시가, 입주연차, 시공사 등 상세 정보
- 호갱노노 단지 페이지로 바로 이동 링크

## 데이터
- 출처: [호갱노노](https://hogangnono.com) `/api/apt/bounding`
- 수집: Playwright(headless Chromium)으로 zoom 16 격자 342타일 순회, XHR 응답 캡처
- 필드: 단지명, 주소, 좌표, 세대수, 전용/공급면적, 실거래 최고가/시점/층, 전세 최고가, 공시가, 입주연차, 거래수, 난방/현관 구조, 시공사

## 파일
- `index.html` — Leaflet 지도 (Vanilla JS, 라이브러리 CDN)
- `data.json` — 9,268개 단지 데이터 (3.4MB, minified)

## 면책 / 주의
- 호갱노노 비공식 데이터로 학습/연구용. 상업적 사용 금지.
- 가격은 **역대 실거래 최고가**라 5년 이상 된 거래도 포함됨. "최근 거래" 필터로 신뢰도 확인 권장.
- 무단 재배포 시 호갱노노 측 요청에 따라 즉시 삭제할 수 있음.
