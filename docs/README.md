# 서울 아파트 지도

서울 25개 자치구 아파트 단지 9,268개를 Leaflet 지도에 시각화.

## 기능
- 가격 · 전용면적 · 세대수 · 자치구 · 거래 시점 필터 (상단 pill 형식)
- 원 색상 = 가격대, 원 크기 = 세대수
- 단지 클릭 시 실거래가, 거래일, 층수, 전세, 공시가, 입주연차 등 표시
